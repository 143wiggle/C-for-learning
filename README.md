# C-for-Learning 

A beginner-friendly C programming journey.  
**Purpose**: Accessible programming code repository for learning C from the ground up.

---

  - `0-hello-world.c`

  - `2-if-elif-else.c`
  - `2a-while-loop.c`
  - `2b-for-loop.c`
  - `2c-range.c`
  - `2d-break-continue.c`
  - `3a-arithmetic-operators.c`
  - `3b-comparison-operators.c`
  - `3c-assignment-operators.c`
  - `3d-logical-operators.c`
  - `4-arrays.c`
  - `5-functions.c`
  - `6-strings.c`
  - `7-pointers.c`
  - `8-structures.c`
  - `9-memory-management.c`
  - `10-file-handling.c`
  - `11-error-handling.c`
  - `12-switch-case.c`
  - `13-nested-loops.c`
  - `14-fgets.c`

  - `16-char-array.c`

- `exercises/`: Practice files to reinforce core concepts.  
  Files:
  - `01-variable-assignment.c`
  - `02-loop-exercises.c`
  - `03-string-manipulation.c`
  - `04-switch-case-exercises.c`
  - `05-nested-loop-exercises.c`

- `projects/`: Small programs combining multiple concepts.  
  Files:
  - `simple-calculator.c`

  - `file-parser.c`

- `assets/`: Visuals and support files.  
  Subfolders:
  - `diagrams/`
  - `resources/`

- `github/`: Repository settings and configurations.  
  Files:
  - `README.md`
  - `LICENSE`
  - `.gitignore`

- `output/`: Program results and builds.  
  Subfolders:
  - `test_outputs/`
  - `compiled_binaries/`

- `docs/`: Study materials and external resources.  
  Files:
  - `C_Study_Guide.md`
  - `C_Programming_Resources.md`

- `Makefile`: Automates compilation of multiple `.c` files.

- `requirements.txt`: Dependencies (optional, for extended C toolchains).

## How to Run

Make sure you have a C compiler like `gcc` installed. Use this in your terminal or command line:

```bash
gcc basic_concepts/1-number.c -o number
./number
